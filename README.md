yichen_toc
=========

### 基于i5ting_toc修改，原作地址[i5ting_toc](https://github.com/i5ting/tocmd.npm)

## 使用

### 安装

```aidl
npm i -g yichen_toc
```

### 生成文档

```aidl
yichen_toc -f README.md -o -s googlecode.css
```

- -f 指定要转换的ｍａｒｋｅｒｄｏｗｎ文档　【必填】
- -o 生成完成后打开　【选填默认不打开】
- -s 指定代码高亮的样式（见下文　highlight 样式）　【选填默认是googlecode.css】
